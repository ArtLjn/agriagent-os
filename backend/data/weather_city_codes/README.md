# 中国城市编号映射表 (weather.com.cn)

## 数据来源

weather.com.cn 使用 101 开头的 9 位数字编码体系：
- 第 1-3 位：101（固定前缀）
- 第 4-5 位：省级行政区编码
- 第 6-7 位：市级行政区编码
- 第 8-9 位：区县编码（00 表示市级）

## 使用方式

**推荐做法（无需此文件）：**
```python
# 优先使用和风天气 GeoAPI
city_info = qweather_geo_lookup("苏州")
city_code = city_info["city_code"]  # 自动获取 101190401
alerts = scraper.fetch(city_code)
```

**Fallback（此文件）：**
```python
import json

with open("city_codes.json") as f:
    CITY_CODES = json.load(f)

code = CITY_CODES.get("苏州")  # "101190401"
```

## 维护说明

此文件为 **fallback 数据**，优先使用和风天气 GeoAPI 动态查询。
仅当以下情况使用此映射表：
1. 和风天气 API 不可用
2. 查询小城镇不在和风天气库中
3. 需要离线运行

## 更新频率

城市编码极少变更，建议每年检查一次即可。
