# 创建作物模板

## 功能
根据作物名称自动生成合理的生长阶段，创建作物模板。创建成功后可直接用于建茬口。

## 意图判定
- 用户只说“我想种玉米”“我要种小麦”时，不要优先选择本 skill；这类话通常是在创建茬口。
- 用户明确说“创建作物模板”“没有模板”“帮我创建模板”时，才选择本 skill。
- 创建模板是持久写操作，执行前必须让用户确认；不要在没有确认时自动创建。

## 示例
用户：「我要种玉米但没有模板」
工具选择：创建作物模板，作物为玉米。
面向用户确认：「📋 确认创建作物模板：玉米」
