# 还赊账 Skill

## 功能
通过对话结清赊账记录，支持部分还款和全额还清两种模式。

## 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| counterparty | string | 是 | 债权人名称/简称，如'老王'、'农资店' |
| amount | number | 否 | 还款金额，不传则全额还清 |
| note | string | 否 | 备注 |

## 使用场景
- 部分还款：「还了老王1000块」→ 查找老王赊账，创建1000元还款记录
- 全额还清：「老王的钱全还了」→ 计算赊账总额，创建等额还款记录

## 数据源
- SQLite 数据库 (CostRecord 表)
