# get_cost_analytics

## 类型
只读(read-only)

## 触发场景
- 用户说"分析一下最近的收支趋势"
- 用户说"今年比去年花了多少"
- 用户说"上个月和这个月的成本对比"
- 用户提到分析、趋势、对比、比去年、比上月时

## 参数
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| date_from | string | 是 | 分析开始日期 YYYY-MM-DD |
| date_to | string | 是 | 分析结束日期 YYYY-MM-DD |
| compare_period | string | 否 | 对比周期: none(不对比)/last_month(上月)/last_year(去年同期)，默认none |

## 返回
```
收支分析 (2026-05-01 至 2026-05-31)：
  本期: 支出 5800.00 元, 收入 12000.00 元, 净 6200.00 元 (15 笔)
    支出TOP3: 化肥(2000), 人工(1500), 农药(800)
  对比期: 支出 4200.00 元, 收入 10000.00 元, 净 5800.00 元 (12 笔)
  对比变化：
    支出变化: +38.1%
    收入变化: +20.0%
```

## 依赖
- `models/cost.py` — CostRecord
- `core/skill_cache.py` — @cached(ttl=300s)

## 错误处理
| 场景 | 返回 |
|------|------|
| 缺少日期范围 | 由 skillify 框架自动校验 |
| 无记录 | 返回零值统计 |

## 示例对话
用户：「分析一下这个月的收支，跟上个月比」
Agent → 调用 get_cost_analytics(date_from="2026-05-01", date_to="2026-05-31", compare_period="last_month")
返回：收支对比数据，含支出/收入变化百分比
