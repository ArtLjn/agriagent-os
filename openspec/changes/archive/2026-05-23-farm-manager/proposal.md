## Why

用户父母承包二三十亩土地，春季种植西瓜、秋季种植豆角，目前完全依靠经验和手工记录管理农事。缺乏系统化的种植周期提醒、天气预警、成本核算工具，导致农事安排容易遗漏、投入产出不清、病虫害防治不及时。需要一个简单易用的数字化工具来提升种植管理效率。

## What Changes

- 开发一款原生移动 App（React Native），面向农民用户的种植管理工具
- 构建 Python（FastAPI）后端服务，提供 RESTful API 和数据管理
- 实现多作物、多茬口的种植周期管理系统，支持自定义生长阶段和提醒
- 集成天气服务，提供灾害预警和农事建议
- 接入 AI 大模型 Agent，结合种植周期、天气、农事记录生成个性化农事建议
- 实现成本记账功能，记录投入产出并生成年度分析报表
- 支持农事打卡记录，形成数字化种植档案

## Capabilities

### New Capabilities

- `crop-cycle-management`: 多作物种植周期管理，支持自定义生长阶段、自动推算全周期节点、推送提醒
- `weather-integration`: 天气数据获取与预警，结合农事场景提供灾害预警和建议
- `ai-farming-agent`: AI 智能农事助手，基于种植周期、天气、农事记录生成个性化建议和待办事项
- `cost-accounting`: 成本记账与统计分析，记录农资、人工、地租等投入和销售收入，生成年度利润报表
- `farm-log`: 农事日志记录，支持快速打卡式记录每日农事操作

### Modified Capabilities

- 无

## Impact

- 新增 React Native 客户端项目
- 新增 Python FastAPI 后端服务
- 新增 SQLite 数据库及数据模型
- 新增第三方天气 API 依赖
- 新增 AI 大模型 API 依赖（用于 Agent 功能）
- 新增推送通知服务（本地通知 + 可选远程推送）
