# Farm Manager 官网 V2 打磨优化计划

## 已发现的问题

### P0 - 严重问题
1. **Hero WebGL流体背景未运行** - 显示纯深绿色静态背景，无交互效果
2. **种植管理区块图片未显示** - 农田航拍图显示为绿色模糊块
3. **App展示3D卡片效果未实现** - 只有1张Mockup滑入，缺少5张卡片的堆叠飞出
4. **AI助手区块标题截断** - "认识芽芽，你的智能农场顾问"被截断/重叠

### P1 - 内容缺失
5. **缺少"使用场景"区域** - 设计要求4个场景（农场主、田间人员、管理者、AI辅助）
6. **功能卡片单调** - 只有图标+文字，缺少配图/screenshot

### P2 - 视觉优化
7. **下载中心背景色刺眼** - #BFFF00 饱和度过高
8. **SVG描边动画未生效** - 文字静态显示
9. **整体动效不够丰富** - 滚动触发动画需要增强

## 优化方案

### Phase A: 核心修复（P0）
- 重写Hero背景为可靠运行的粒子/流体效果
- 修复种植管理区块为图片+遮罩渐变
- 重新实现App展示为CSS 3D变换卡片
- 修复AI助手区块布局

### Phase B: 内容增强（P1）
- 新增使用场景区域（4场景卡片轮播）
- 功能卡片添加配图网格

### Phase C: 视觉打磨（P2）
- 优化下载中心配色
- 增强FAQ交互动画
- 添加更多滚动触发动画
- 优化响应式细节
