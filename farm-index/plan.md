# Farm Manager 官网项目计划

## 项目概述
为 Farm Manager（智慧农场管理平台）构建一个高质量、产品级的官方网站。
- 中文为主，英文品牌名保留
- 视觉风格：现代、干净、农业科技感、自然不土气
- 配色：绿色、蓝青、白色、浅灰
- 响应式：桌面 + 移动端

## 技术栈
- React + TypeScript + Vite
- Tailwind CSS
- 无后端依赖

## 页面结构与区块
1. Hero 首屏 - 强产品识别度 + CTA
2. 产品亮点 - 6大核心功能卡片
3. 使用场景 - 4个用户场景
4. 产品截图展示 - 移动端 UI Mockup
5. 下载中心 - 完整可维护的下载区域
6. FAQ - 6个常见问题
7. 页脚 - 完整导航与占位信息

## 执行阶段

### Stage 1 — 设计与素材生成
- 加载 vibecoding-webapp-swarm 技能
- 生成 Hero 背景图、产品截图 Mockup、AI助手形象等视觉素材
- 产出 design.md

### Stage 2 — 网站开发
- 初始化项目
- 实现所有区块组件
- 响应式适配
- 交互动效

### Stage 3 — 构建与部署
- 构建生产版本
- 部署上线

## 质量标准
- 接近成熟 SaaS 产品官网水平
- 首屏必须有强产品识别度和下载转化能力
- 代码结构清晰，方便后续替换下载链接
