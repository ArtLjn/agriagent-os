# 首页 UI Spec

## Image

- Source image: `/Users/ljn/Documents/demo/explore/output/imagegen/mvp-pages/home.png`
- Canvas: `1024x2224px`
- Page: `首页`

## Purpose

首页只回答“今天先看什么、先做什么”，避免管理后台式指标堆叠。

## Required UI

- Top: `今日`, `今天先看这几件事`
- Card: `今天该做什么`
- Card: `降温提醒`
- Card: `待确认 2条`
- Card: `最近记录`
- Bottom nav: `首页`, `记录`, `芽芽`, `账本`, `我的`; `首页` selected.

## Invariants

- 不放复杂经营仪表盘。
- 不放密集功能入口。
- 重点是提醒、待确认、最近记录。
