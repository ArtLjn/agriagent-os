# Farm Manager Flutter App

## 运行

```bash
flutter run
```

## 测试

```bash
flutter test
```

## 视觉验收

- 五个 Tab：首页、工作台、芽芽、账单、我的。
- 芽芽页面和底部 Tab 统一叫“芽芽”。
- 页面 UI 不展示任何 API 路径。
- 芽芽主色使用蓝青系，不使用紫色。
