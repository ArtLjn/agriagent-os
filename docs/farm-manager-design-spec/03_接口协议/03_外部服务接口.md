## 1. LLM Provider

### 1.1 OpenAI-compatible Provider（主）

| 项 | 值 |
| --- | --- |
| 默认端点 | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| 默认模型 | `qwen3.6-35b-a3b` |
| 备选 | `providers.json` 中按 provider/model/role/priority 配置 |
| 认证 | OpenAI-compatible `Authorization: Bearer <AI__API_KEY>` |
| 配置入口 | `backend/app/shared/config.py` 的 `AIConfig`，或仓库根 `providers.json` |
| 超时 | 30s |
| 重试 | 按 `CIRCUIT_BREAKER__RETRY_MAX` 与熔断 cooldown 策略 |

**关键能力**：
- Tool Use（function calling）
- 多 Provider / 多 Key 轮转与 fallback
- Streaming（SSE）
- `enable_thinking` 可配置

**封装位置**：`backend/app/shared/llm.py`

### 1.2 Provider 路由与兜底

| 来源 | 用途 |
| --- | --- |
| `providers.json` | 多 provider、多模型、角色、优先级、权重、key 池配置 |
| `config.yaml` / 环境变量 | 无 `providers.json` 或加载失败时的单 Provider 兜底 |
| `AIConfig` 默认值 | DashScope OpenAI-compatible endpoint + `qwen3.6-35b-a3b` |

`LLMClientManager` 返回 LangChain `ChatOpenAI`、OpenAI 同步客户端或异步客户端，调用方不直接绑定具体模型厂商。

### 1.3 接入方式

```python
class LLMClientManager:
    """根据 providers.json / config.yaml 路由到具体 Provider。"""

    def get_chat_model(self, *, role: str = "generation") -> ChatOpenAI: ...
    def get_sync_client(self, *, role: str = "generation") -> OpenAI: ...
    def get_async_client(self, *, role: str | None = None) -> AsyncOpenAI: ...
```

### 1.4 dfm-openai-proxy（自建代理）

- 公网部署，token 通过环境变量 `DFM_PROXY_TOKEN` 注入，禁止写入文档或代码仓库
- 用途：国内访问 OpenAI 兼容接口的代理
- 用于：OpenAI 兼容的国产 Provider 接入

## 2. 天气服务

### 2.1 QWeather（主）

| 项 | 值 |
| --- | --- |
| 端点 | `https://p32k5pxvta.re.qweatherapi.com/v7` |
| 认证 | API Key（query 参数）或 appid/appsecret 签名 |
| 主要接口 | `/weather/now`、`/weather/{days}d`、`/air/now` |
| 城市定位 | 经纬度优先；城市名通过 `https://p32k5pxvta.re.qweatherapi.com/v2/city/lookup` |
| 配额 | 免费 1000 次/天，付费 10 万次/天 |
| 缓存 | 5 分钟（current）、1 小时（forecast） |

### 2.2 Open-Meteo（兜底）

| 项 | 值 |
| --- | --- |
| 端点 | `https://api.open-meteo.com/v1` |
| 认证 | 无（免费） |
| 主要接口 | `forecast`（含 current + hourly + daily） |
| 优势 | 不需 API Key，全球覆盖 |
| 缺点 | 无农业建议、无中文描述 |

### 2.3 双源策略

```python
async def fetch_weather(
    location: str = "",
    days: int = 7,
    lat: float | None = None,
    lon: float | None = None,
) -> dict:
    strategy = get_weather_strategy()
    data = await strategy.fetch(location, days, lat, lon)
    return _to_legacy_format(data)
```

封装位置：`backend/app/domains/weather/service.py`、`backend/app/domains/weather/providers/`

### 2.4 农业天气预警

当前 `check_weather_warnings()` 会优先使用 provider 返回的 warnings；没有官方预警时，按温度、降水、风速阈值生成基础农事预警。DataFlywheel 可在后续基于历史预警 + 农户茬口生成「该不该浇水/打药/采收」建议。

## 4. MySQL（业务数据）

| 项 | 值 |
| --- | --- |
| 版本 | 8.x |
| 字符集 | utf8mb4 / utf8mb4_unicode_ci |
| 引擎 | InnoDB |
| 主备 | 生产环境主备（异步复制） |
| 连接串 | `mysql+pymysql://user:pass@host:3306/db?charset=utf8mb4` |
| 连接池 | SQLAlchemy QueuePool，size=10，max_overflow=20 |

**禁止**：
- 手动 SQL 改表（用 Alembic）
- 字符集非 utf8mb4
- 跨 farm_id 查询（必须带 farm_id WHERE）

## 6. systemd 服务

```ini
# /etc/systemd/system/<service_name>.service
[Unit]
Description=Farm Manager Backend
After=network.target mysql.service

[Service]
Type=simple
User=<deploy_user>
WorkingDirectory=<repo_dir>/backend
ExecStart=<repo_dir>/backend/.venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000
Restart=always
RestartSec=3
Environment=PYTHONUNBUFFERED=1

[Install]
WantedBy=multi-user.target
```

具体路径、用户名、服务名不在设计 spec 中暴露，由部署仓库管理。

## 7. Nginx（反向代理）

```nginx
server {
    listen 443 ssl http2;
    server_name farm.example.com;

    ssl_certificate /etc/letsencrypt/live/farm.example.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/farm.example.com/privkey.pem;

    # Admin Web 静态资源
    location / {
        root /var/www/farm-admin;
        try_files $uri /index.html;
    }

    # API：后端当前直接暴露 /auth、/agent、/smart-fill、/planting、/admin 等 root 路由
    location ~ ^/(auth|agent|smart-fill|crops|cycles|logs|planting|costs|cost-categories|debts|weather|locations|admin|simulation|api/app)/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Trace-Id $http_x_trace_id;
    }

    # SSE 长连接
    location /agent/chat/stream {
        proxy_pass http://127.0.0.1:8000;
        proxy_http_version 1.1;
        proxy_set_header Connection "";
        proxy_buffering off;
        proxy_read_timeout 600s;
    }
}
```

## 8. 环境变量清单

| 变量 | 用途 | 必填 |
| --- | --- | --- |
| `DATABASE__URL` | MySQL 连接串 | ✅ |
| `AI__API_KEY` | OpenAI-compatible API Key | ✅ |
| `AI__BASE_URL` | OpenAI-compatible endpoint，默认 DashScope | ❌ |
| `AI__MODEL` | 模型名，默认 `qwen3.6-35b-a3b` | ❌ |
| `SECRETS__QWEATHER_API_KEY` | 和风天气 Key | ✅ |
| `SECRETS__QWEATHER_APPID` / `SECRETS__QWEATHER_APPSECRET` | 和风天气签名认证 | ❌ |
| `AUTH__JWT_SECRET` | JWT 签名密钥 | ✅ |
| `DFM_PROXY_TOKEN` | dfm-openai-proxy token | ✅ |
| `LANGSMITH__API_KEY` | LangSmith（可选） | ❌ |
| `LOG_LEVEL` | 日志级别（默认 INFO） | ❌ |
| `TOKEN_QUOTA__MONTHLY_LIMIT` | 用户月 Token 配额 | ❌ |

所有密钥从 `.env` 加载，**禁止入库 git**。
