设计说明

主入口

左上角菜单打开会话侧边栏，右上角加号直接新建会话。这样既参考了你给的 App， 又不会把历史列表塞进主聊天页面。

抽屉结构

顶部放搜索和新建；中部按“今天、7 天内、30 天内”分组；底部放当前用户和设置。 侧边栏宽度控制在 304px，右侧保留当前页面预览和遮罩。

农业场景标题

会话标题建议自动生成，配一个轻量标签：天气、种植、病虫害、报告、记账。 用户扫历史时能马上知道这轮对话的用途。

交互建议

点常用场景时默认创建新 session；如果当前 session 已有未发送输入，则弹确认。 从历史进入后，输入框和上下文都绑定对应 session。
