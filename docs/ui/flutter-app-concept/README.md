# Flutter App Concept

本目录保存 Farm Manager Flutter App 的移动端视觉方向预览。

- `index.html`: 五个核心 Tab 的高保真静态预览。
- 设计方向: 首页是芽芽信息聚合页，工作台是全功能快捷导航，芽芽是可交互聊天，账单是核心经营页。
- 视觉约束: 不走土味农业风，不大面积使用绿色，不堆叠叶子、稻穗、田园插画。

## 页面设计意图

- 首页: 一个大而全的芽芽聚合卡片，集合今日建议、天气、作业和人工结算提醒。
- 工作台: 类似钉钉/支付宝的功能宫格，是后续业务跳转的统一入口。
- 芽芽: 可交互聊天页，左上角菜单用于打开历史对话侧边栏。
- 账单: 核心页面，突出本月收支、快速记账、成本构成和最近流水。
- 我的: 参考成熟产品的个人中心结构，聚合账号、设置、帮助与系统入口。

## 页面与后端能力对应

- 首页: 经营汇总、天气预报、作业提醒、未结人工。
- 工作台: 批次管理、作业管理、种植单元、工人管理、作业类型、农事日志。
- 芽芽: 聊天、历史会话、经营报告、上下文建议。
- 账单: 成本流水、年度汇总、批次利润、欠款管理。
- 我的: 账号信息、个人设置、应用版本。
