# FarmManager Mobile UI v2 设计文档

> 基于 2026-05-27 Mobile UI Redesign 计划实现

---

## 设计哲学

从传统管理工具风格（深青色、密集布局）转型为 **AI 助手设计语言**：
- **配色**：蓝紫绿三色体系，柔和渐变替代纯色块
- **留白**：大卡片、圆角、充足间距
- **动效**：呼吸浮动、淡入上滑、点击缩放，营造"会呼吸"的界面
- **拟人**：AI 小宠物悬浮按钮、AI 头像、情绪化天气卡片

## 动画规范

| 动画 | 持续时间 | 参数 | 用途 |
|------|----------|------|------|
| 淡入上滑 | 450ms | translateY: 20→0, opacity: 0→1 | 卡片入场 |
| 呼吸浮动 | 3000ms | translateY: ±4px 循环 | AIPet 悬浮 |
| 点击缩放 | 100ms | scale: 1→0.96 | 按钮按压反馈 |
| 交错延迟 | 80ms | 每卡片递增 | 首页多卡片入场 |
