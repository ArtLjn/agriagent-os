# 登录注册 Flutter 复刻规格

## 目标

登录、注册、首次设置页面沿用主 App 参考风格，不做营销落地页，不做复杂后台表单。

## 视觉方向

- 登录注册需要比普通表单更有品牌感。
- 使用柔和抽象品牌区域、圆角浮层、蓝绿轻量色块，避免单纯白卡片堆叠。
- 不要农田、农作物、机械、暗色后台或工业风。
- 认证页可以比主功能页更留白、更有 onboarding 感。

## 页面

### 登录

组件顺序:

1. 顶部品牌区:
   - `农场管家`
   - `把农场记录得更轻松`
   - 抽象圆角蓝绿品牌图形，不能用复杂农田插画。
2. 表单卡片:
   - `手机号`
   - `密码`
   - `忘记密码`
3. 主按钮:
   - `登录`
4. 注册入口:
   - `还没有账号？去注册`

### 注册

组件顺序:

1. 标题:
   - `创建账号`
   - `先建账号，记录可以慢慢补`
2. 表单卡片:
   - `手机号`
   - `设置密码`
   - `昵称`
3. 主按钮:
   - `注册并进入`
4. 登录入口:
   - `已有账号？去登录`
5. 说明:
   - `MVP功能全部免费`
6. 轻量能力标签:
   - `手动记录`
   - `AI帮填`
   - `账本提醒`

### 首次设置

注册后首次进入，用于补充农场信息。

组件顺序:

1. 标题:
   - `完善农场信息`
   - `这些信息之后也可以修改`
2. 抽象能力说明区:
   - `天气`
   - `记录`
   - `账本`
3. 表单卡片:
   - `农场名称`
   - `所在城市`
   - `默认天气城市`
   - `身份`: 默认 `农场负责人`
4. 主按钮:
   - `开始使用`
5. 次级入口:
   - `稍后再说`
6. 提示:
   - `可稍后在我的中修改`

## 设计 Token

- 背景: `#F7F9FC`
- 卡片: `#FFFFFF`
- 主蓝: `#2F73F6`
- 绿色: `#35C879`
- 橙色: `#FF9F1C`
- 正文: `#111827`
- 次级文字: `#6B7280`
- 边框: `#E8ECF2`
- 页面横向 padding: `24px`
- 卡片圆角: `24px`
- 输入框高度: `52px`
- 主按钮高度: `52px`
- 主按钮圆角: `14px`
- 顶部品牌区域高度: `230-300px`
- 表单卡片 padding: `20px`

## 实现约束

- 不要农田、农作物、叶子、土地、机械插画。
- 不要暗色工业风。
- 表单字段不要过多。
- 触达区域不小于 `48px`。
- 登录注册可先做静态 UI，但字段命名要和后端认证语义对应: 手机号、密码、昵称。
