屏幕设计说明

首页首屏只放当天关键判断：城市、天气、AI 建议、四个快捷动作。天气卡是视觉峰值，快捷入口保持 4 个以内。
芽芽对话页保留会话抽屉和新建入口；报告页使用同一 segmented control，避免二级 Tab 膨胀。
记账金额卡占据上半屏注意力，筛选 chip 直接暴露；新增按钮固定拇指区。
种植/工人列表项强调作物、状态、阶段、面积、单元和进度；工人页强调未结金额与快速记工资。

组件规则

导航底部 4 Tab：首页、芽芽、记账、我的；业务入口从首页和我的页双入口进入。
表单优先选择组件，减少手输；金额、日期、工人、茬口使用专用选择器。
列表每项左侧图标色块，中间主标题+上下文，右侧金额或状态；支持长按批量选择。
AI 执行确认AI 识别出记账/记工资/建茬口等操作时，消息卡底部出现确认/取消按钮。

状态覆盖

空茬口“新建第一个种植批次” CTA。
空账单展示记账入口与示例分类。
天气失败使用缓存天气，显示更新时间。
AI 超时保留输入，显示重试和转手动入口。

开发交付提示

适配基线：Android 360-430dp 宽度，状态栏深色，内容使用 SafeArea bottom。
复用文件：颜色来自 `src/theme/colors.ts`，圆角/字号来自 `spacing.ts` 与 `designTokens.ts`。
图标：沿用 MaterialCommunityIcons；按钮优先图标+可访问标签。
验证：检查 375px 小屏中文不截断，FAB 不遮挡末条记录，键盘弹起后输入栏可见。
